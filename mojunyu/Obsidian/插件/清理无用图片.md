
## 作用

自动清理没用过的笔记图片，省硬盘空间

## 设置


| 设置项                             | 推荐配置                      | 说明                                                          |
| ------------------------------- | ------------------------- | ----------------------------------------------------------- |
| **Ribbon Icon**​                | 开启                        | 在左侧导航栏显示清理图标，方便一键操作。                                        |
| **Deleted Image Destination**​  | **Move to System Trash**​ | **安全第一**。建议选“系统回收站”或“Obsidian回收站”，避免误删后无法恢复。                |
| **Excluded Folder Full Paths**​ | 填写路径                      | **重要**：填写你存放“封面图”或“素材库”的文件夹路径（如 `assets/images`），防止有用图片被误删。 |
| **Exclude Subfolders**​         | 开启                        | 排除路径时，是否同时排除其子文件夹。                                          |
| **Delete Logs**​                | 开启                        | 清理后弹出日志窗口，显示删除了哪些文件，便于核对。                                   |
|                                 |                           |                                                             |
|                                 |                           |                                                             |


## 使用

1. **图标点击**：开启 Ribbon Icon 后，直接点击左侧栏的“清理”图标。
    
2. **命令面板**：按 `Ctrl+P`（Win/Linux）或 `Cmd+P`（Mac），输入 `Clear Unused Images`并运行。
    

插件会自动扫描所有 Markdown 文件中的链接，将未被引用的图片移动到你在设置中指定的位置（如回收站）。
