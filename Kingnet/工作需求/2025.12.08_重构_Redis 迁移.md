# Redis 迁移 - king-ad-redis

## 迁移目标

将 `king-ad-redis` 迁移至新加坡 Redis，并清理废弃的队列和配置。

---

## 迁移步骤

| 步骤 | 操作 | 说明 |
| ---- | ---- | ---- |
| 1 | 迁移后台使用的 Redis 数据库 key | 迁移至新加坡 Redis，确认无问题后删除后台配置 |
| 2 | 检查 SDK 使用情况 | 清理代码，删除配置 |
| 3 | 关闭 Redis | 等待退还资源 |
| 4 | 升级新加坡 Redis 配置 | 升级为 4G |
| 5 | 清除 MySQL 只读节点 | 清除香港、台湾节点的只读机器 |
