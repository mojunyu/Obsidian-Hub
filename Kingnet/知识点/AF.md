# AppsFlyer 知识点

---

## Q1: 如何计算包体所有渠道（包括自然量）的 ATT 通过率？

**问**：如果需要计算包体所有渠道包括自然量的 ATT 通过率，是不是应该将"自然"和"非自然"两份报告合并，计算总的 IDFA 比例？

**答**：是的，需要合并计算。

总 ATT 通过率 = (自然 IDFA 用户数 + 非自然 IDFA 用户数) / (自然总用户数 + 非自然总用户数)

也可以分渠道单独查看各渠道的 ATT 通过率，便于对比分析。
